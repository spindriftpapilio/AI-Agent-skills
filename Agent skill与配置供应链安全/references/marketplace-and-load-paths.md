# 市场与加载路径

## 公开线索

- Microsoft公开了`microsoft/skills`仓库，说明skills、MCP配置与项目级Agents正在成为正式生态对象，而不只是零散提示文件。
- 一些本地聚合型MCP项目会自动收集多个目录中的skills并暴露给Agent使用。

这类生态意味着审查重点要从单文件安全，扩展到：

- 来源可信性
- 自动发现路径
- 版本与更新策略
- 目录优先级覆盖
