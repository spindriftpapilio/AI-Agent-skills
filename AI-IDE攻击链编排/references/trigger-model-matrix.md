# 触发模型矩阵

按产品记录什么样的用户动作会触发什么样的链。评估时可直接在下面补充。

# 触发模型定义

|模型|用户动作|例子|
|---|---|---|
|零交互|无额外动作|打开工作区即自动加载配置|
|单次交互|用户做一次正常操作|解释这段代码触发提示词注入|
|Agent自动运行|用户开启agent模式|Agent自主扫描与执行|
|延时触发|初次之后无需再点|先批准，后续内容被替换|
