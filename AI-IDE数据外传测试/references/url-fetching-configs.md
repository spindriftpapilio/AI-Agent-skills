# 预配置URL拉取向量

这类问题主要看：工作区配置会不会让IDE主动去抓远程地址。

# VS Code、VS Code系产品

### 远程JSON Schema**常用配置位置：**

- `.vscode/settings.json`

**基本思路：**
- 某些设置允许指定远程schema、文档源或资源地址
- 打开相关文件或触发校验时，IDE会主动请求该地址

# 可重点留意的配置类型

- JSON schema URL
- 模型提供方、API地址
- 扩展下载源
- 文档、图片、模板、snippet远程地址

# 验证流程

1. 先用固定地址看是否真的会请求
2. 再看能否把工作区信息或本地数据拼进URL
3. 最后区分请求是打开即发生，还是某个动作触发
